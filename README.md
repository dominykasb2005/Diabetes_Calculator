# Diabetes_Calculator
Simple python machine learning algorithm that reads data from a .csv file about diabetes patients, and can give a user an idea on whether the patient symptoms they fill in, could have diabetes. 

TO RUN:
1 -> Place all 3 files in same folder, run "DiabetesGUI" 
2 -> Once GUI loads, press "TRAIN" and then "RUN MODEL" under model controls panel
3 -> Enter desired symptoms under patient details panel and press "predict result"
